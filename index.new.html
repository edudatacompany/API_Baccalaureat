API Résultats Baccalauréat

Démo technique pour le Ministère de l'Éducation nationale

Bienvenue sur la plateforme de test

Ce projet expérimental vise à exposer une API REST simulée, permettant un accès structuré aux données (actuellement fictives) des résultats du baccalauréat.

Les endpoints disponibles permettent la recherche par académie, lycée ou élève :

  • /api.php?endpoint=academie&nom=paris
  • /api.php?endpoint=lycee&id=101
  • /api.php?endpoint=eleve&id=2024_001

Les données affichées sont entièrement fictives et ne représentent en aucun cas des données réelles sur ce prototype.
